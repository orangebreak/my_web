Technical Skills

  • Python A versatile high-level language known for its readability and broad applications in data science, web development, and automation.
  • C A powerful low-level programming language that provides direct memory access and is fundamental for system programming and embedded systems.
  • C++ An extension of C with object-oriented features, used for high-performance applications, game development, and system software.
  • HTML HyperText Markup Language is the standard markup language for creating web pages and web applications.
  • CSS Cascading Style Sheets is used for describing the presentation of a document written in HTML, controlling layout, colors, and animations.
  • JavaScript A high-level, interpreted programming language that enables interactive web pages and is essential for modern web development.
